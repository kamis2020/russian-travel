# **Путешествие по России** 
## Проектная работа №3 от Яндекс Практикума.
### Адаптивный одностраничный сайт про путешествие по России. 

#### Технологии использованные при верстке проекта:
* Файловая структура БЭМ (Nested)
* Flexbox верстка
* Grid верстка
* Медиа-запросы (@media)
